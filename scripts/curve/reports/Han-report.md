使用新数据生成方法生成了单圈与双圈数据各1万个。双圈数据之压缩方法为对x方向做压缩变换:
$$x' = (x-2.5) * 0.49 + 2.5; x' = (x+2.5) * 0.49 - 2.5 $$
目前我之结果:皆为四种核在单圈上做训练(训练集大小9千)并在双圈上做测试(测试集大小1千)

四种核:
1. sp_laplace: 单层势
2. dp_laplace: 偶极子势
3. modified_laplace: 去掉了与法向之点积的二维值偶极子势卷积核
4. stokes: stokes核

诸参数方面:
1. 四种核中唯有dp_laplace需要"normal_prod=True"
2. 每个测试的四种参数设定与记录名称:
   
   | grad | geo | lap | name |
   | :--- | :--- | :--- | :--- |
   | true | false | false | noact |
   | true | true | false | noact_geo3wx |
   | false | true | false | noact_nograd_geo3wx |
   | false | false | false | noact_nograd |
