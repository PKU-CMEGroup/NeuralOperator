## Features
The features variable has the shape (ndata, nnodes, 9), which consists of 9 dimensions, where each dimension represents specific information:

Output
- 0 mean pressure of neighbour triangles

Infos
- 1 length
- 2 width
- 3 height
- 4 clearance
- 5 slant
- 6 radius
- 7 velocity
- 8 reynolds number
